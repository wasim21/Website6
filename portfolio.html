Face mask detection using deep learning

In response to the global health crisis brought on by COVID-19, mask-wearing has become a critical public health practice. To assist in monitoring compliance, this project aims to build an AI-based face mask detection system using deep learning techniques. The system is designed to identify individuals who are and are not wearing masks in real-time from camera feeds, providing quick and accurate feedback to help authorities and organizations monitor safety protocols.
